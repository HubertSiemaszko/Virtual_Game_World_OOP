# Simple Virtual World Simulator

A simple console-based simulation project written in C++. It creates a grid world where different animals and plants coexist, move, and interact with each other turn by turn.

## How to play
You control the Human character. Use the keyboard to determine your next move and watch how the world evolves.

### Controls
* **Arrows / W A S D** - Move Human
* **U** - Activate Special Ability
* **Z** - Save game state
* **L** - Load game state
* **Q** - Quit
